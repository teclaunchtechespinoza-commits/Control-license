<product_requirements>
The project aims to build a comprehensive Complete License Management System from a React/FastAPI template. Key requirements include secure multi-user functionality with JWT authentication and role-based access (Admin/User). Core features encompass CRUD operations for licenses, users, and Registry Modules (Categories, Companies/Clients, Products, License Plans). A detailed Client Management system is crucial, supporting Pessoa Física (PF) and Pessoa Jurídica (PJ) clients with extensive fields for identification, contact, address, billing, LGPD compliance, audit trails, and technical data. Current implementation includes a functional backend, a semantic color system for WCAG accessibility, a frontend version control display, and sensitive data masking for client information. The system is progressively stable, with recent efforts focused on login bug fixes and the initiation of RBAC.
</product_requirements>
